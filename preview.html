安装和运行步骤：

  1. 下载Node.js：访问 https://nodejs.org 下载LTS版本
  2. 安装Node.js：双击.msi文件，按照向导完成安装
  3. 验证安装：打开命令提示符，输入 node --version
  4. 安装依赖：在项目目录运行 npm install
  5. 启动服务器：运行 npm run dev
  6. 访问网站：浏览器打开 http://localhost:3000
