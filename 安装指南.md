# Node.js和npm安装指南

## 安装步骤

### 1. 下载Node.js
- 访问 https://nodejs.org/
- 下载LTS（长期支持）版本
- 选择Windows安装包（.msi格式）

### 2. 安装过程
1. 双击下载的.msi文件
2. 点击"Next"继续
3. 勾选"I accept the terms in the License Agreement"
4. 点击"Next"（保持默认安装路径）
5. 点击"Next"（保持默认配置）
6. 点击"Install"开始安装
7. 安装完成后点击"Finish"

### 3. 验证安装
安装完成后，打开新的命令提示符或PowerShell窗口，输入：

```cmd
node --version
npm --version
```

如果显示版本号，说明安装成功。

## 常见问题解决

### 问题1：命令无法识别
如果提示"node不是内部或外部命令"，请：
1. 重新启动命令提示符
2. 检查环境变量是否配置正确
3. 可能需要重启计算机

### 问题2：权限问题
如果遇到权限错误，请：
1. 以管理员身份运行命令提示符
2. 或使用PowerShell管理员模式

### 问题3：端口被占用
如果3000端口被占用，Vite会自动选择其他端口
